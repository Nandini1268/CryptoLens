# CryptoLens 🔍
A Chrome Extension that shows live cryptocurrency prices on hover — on any website.

## Features
- 💰 Live prices on hover
- 📈 7-day price chart
- 🔔 Price alerts
- 🌙 Dark mode
- 💱 USD / EUR / INR support

## Tech Stack
HTML + CSS + JavaScript · Chrome Manifest V3 · CoinGecko API

## How to Install
1. Download the ZIP
2. Go to chrome://extensions
3. Enable Developer Mode
4. Click Load Unpacked → select this folder
